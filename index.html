Mémo GitHub



creer fichier :
touch + nom

initialiser sur git :
git init

connextion :
git config --global user.name 'nom'

°°°git config --global user.email 'mail'

présave un fichier :
git add + nom

save :
git commit (VIM : i + commentaire + échap + :wq)

°°°(sans VIM : git commit -m 'commentaire')

voir les saves :
git status

retier une save :
git rm --cached + nom

présave tout le dossier :
git add .

présave tout fichiers dossier par extention :
git add *.ext *.ext [...]

ctrl z une suppretion manuelle :
git checkout

liste ignoré :
touch .gitignore ((nom fichier + /) dans l'éditeur)

créer branche :
git branch + nom

switch de branche :
git checkout + nom de la branche

liste des branches :
git branch --list

fusionner 2 branches :
git merge + nom de la branche à fusionner

Ajouter fichier au repo après commit :
git push

Récuperer changements d'un membre externe :
git pull

Paste un repo sur le PC :
git clone + lien des fichiers du repo
